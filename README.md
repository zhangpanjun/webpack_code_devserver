# webpack_code_devserver
基于webpack5，在使用webpack_devServer，打开开发服务的一些配置，本地代码保存后无需每次打包，页面热更新直接看到最新资源
# node版本
当前运行版本 18.15.0 
# 命令
下载依赖 yarn
开启工程 yarn dev
打包 yarn build
# 注意！！！
devServer是开发服务器，打包后配置就不可用啦，要是访问proxy代理资源，记得配置ngnix呀
